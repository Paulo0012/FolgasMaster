# FolgasMaster
<img width="1917" height="637" alt="image" src="https://github.com/user-attachments/assets/137d20ec-e37f-41b4-9541-0c836fd524a9" />


Sistema web desenvolvido com Django para gerenciamento de servidores, equipes, afastamentos, horas extras e folgas compensatórias.

O projeto foi estruturado seguindo boas práticas do framework Django, utilizando views baseadas em função (FBV) e views baseadas em classe (CBV), autenticação nativa e banco de dados SQLite para facilitar a execução local.

 Funcionalidades:

* Autenticação de usuários (login e logout)
* Dashboard principal
* Cadastro, edição e exclusão de servidores
* Listagem de equipes
* Registro de afastamentos
* Registro de horas extras
* Cálculo e controle de folgas compensatórias
* Upload e exibição de arquivos de mídia

Tecnologias Utilizadas:

Python 3.11+

Django 5.2

SQLite3

HTML5

CSS3 (Bootstrap)
